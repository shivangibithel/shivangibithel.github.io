Beat

Beat is a music player that I developed as part of my summer training at CDAC Mohali. The application was developed using Android Studio in JAVA. It uses the shared content data of the android phone and plays the mp3 file chose by the user. The application has following features:

  • Play and pause song
  • An option to record the audio
  • Searching lyrics for any song
  • Playlist creation
  • Next and previous buttons to change songs
